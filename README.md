# OussamaShoesAI-Studio
OussamaShoes AI 👟✨ — A 3D sneaker customization platform powered by AI. Transform a 2D image (PNG) into a realistic 3D model in one click, and customize it using an advanced layering system: magic brush, sticker placement, and paint splash effects. Built with Next.js 16 and React Three Fiber.
OussamaShoes AI — 3D Sneaker Customization Studio 👟🚀

This tool is specifically designed for sneaker customization artists as well as brands looking to deliver an immersive customer experience.

✨ Key Features
🔄 2D to 3D Conversion (Smart Extrusion)
Import a side-view image (PNG/JPG) of a shoe. Our algorithm instantly generates a volumetric 3D model (realistic inflated effect) that can be viewed and manipulated in 360°.
🪄 AI-Powered Selection (Magic Wand & Lasso)
Precisely isolate parts of the 3D model such as laces, sole, or upper with a single click using a Flood Fill-based algorithm.
🎨 Smart Painting & Splatter Effects (Splatter FX)
A physics-based paint simulation system that projects realistic paint splashes onto the shoe geometry — a must-have for customization artists.
🏷️ Dynamic Sticker System
Apply any logo (e.g., Nike, Adidas, or custom brands) directly onto the 3D model. Resize, rotate, and anchor them in real time.
📱 Optimized Mobile-First Interface
A premium, immersive UI inspired by ergonomic tab systems (Style Panel, Tool Drawer, and AI Panel), fully maximizing the 3D canvas visibility.
📦 Native GLB/GLTF Export
Instantly export your customized creation in binary or JSON format for use in Blender or production pipelines.
🛠️ Tech Stack & Architecture
Frontend Core: Next.js 16 (App Router) / React 19
3D & WebGL Rendering: Three.js, @react-three/fiber, @react-three/drei
State Management: Zustand (ensuring perfect synchronization between UI and the 3D canvas)
Styling: Pure CSS with a premium design approach (immersive dark mode, glassmorphism, smooth transitions)
